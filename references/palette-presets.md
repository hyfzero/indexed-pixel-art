## Four-Color Slime

```json
[
  { "index": 0, "hex": "#E8F6D0", "name": "background" },
  { "index": 1, "hex": "#1E3A2A", "name": "outline" },
  { "index": 2, "hex": "#71C837", "name": "body" },
  { "index": 3, "hex": "#B8F05A", "name": "highlight" }
]
```

Freeze the selected palette before writing pixels. Do not approximate, quantize, or append colors
during drawing or revision.
